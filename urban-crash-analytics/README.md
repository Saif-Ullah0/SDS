# Urban Crash Analytics

A data science project for analyzing urban traffic crash patterns and safety insights.

## Project Structure

- `data/`: Data files
  - `raw/`: Raw data files (not tracked in git due to size)
  - `processed/`: Processed/cleaned data
  - `external/`: External data sources
- `notebooks/`: Jupyter notebooks for analysis
- `src/`: Source code
- `outputs/`: Generated outputs (figures, models, maps)

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run notebooks in order for analysis workflow

## Note

Raw data files are excluded from version control due to size constraints. Place your data files in `data/raw/` locally.
